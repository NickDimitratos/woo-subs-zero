# Security Policy

## Supported Versions

Security updates are provided for the latest release line in this repository.

## Reporting a Vulnerability

Please do not disclose vulnerabilities in public issues.

Use one of these private channels:

- GitHub Security Advisory for this repository.
- Direct contact to repository maintainers.

When reporting, include:

- Affected version and environment details.
- Reproduction steps or proof of concept.
- Expected impact and any suggested mitigation.

## Response Expectations

- Initial triage target: within 3 business days.
- Status updates provided during investigation and remediation.
- Public disclosure coordinated after a fix is available.
